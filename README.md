# Layout

## header 

Define la cabecera de una sección o página.

## nav 

Define una sección de navegación.

## main 

Indica el contenido principal de la página.

## section 

Define una sección temática del contenido.

## article 

Representa un contenido independiente y autónomo.

## aside 

Contiene contenido relacionado o complementario al contenido circundante.

## footer 

Representa el pie de una sección o página.

## figure 

Utilizada para encapsular contenido multimedia.

# Text

## h1 a h6 

Encabezados de nivel 1 a 6, donde h1 es el más importante y h6 el menos importante

## p 

Define un párrafo de texto.

## span

Utilizada para aplicar estilos a un texto o crear pequeñas secciones de texto sin un significado específico.

## strong

Resalta el texto en negrita.

## em

Resalta el texto en cursiva o énfasis.

# Forms

## form

Crea un formulario que agrupa elementos de entrada.

## input

Define un campo de entrada, como texto, botón, checkbox, radio, etc.

## textarea

Crea un campo de entrada de texto de múltiples líneas.

## Label

Etiqueta para describir un elemento de formulario.

## select

Crea una lista desplegable de opciones.

## option

 Define una opción en una lista desplegable **select**

## button

Crea un botón en un formulario.

# Lists

## ul

Define una lista desordenada.

## ol

Define una lista ordenada.

## li

Define un elemento de lista ordenada o desordenada.

## dl

Define una lista de descripción.

## dt

Define un término en una lista de descripción dl

## dd

Define una descripción para un término dt  en una lista de descripción dl

# Media

## img

Inserta una imagen en la página.

## audio

Inserta contenido de audio, como música o efectos de sonido.

## video

Inserta contenido de video.

# Navigation

## a

Define un hipervínculo a otra página web, archivo o recurso.

## link

Define la relación entre el documento actual y un recurso externo, como una hoja de estilo CSS 

# Style & Format

## style 

Permite incrustar reglas de estilo CSS en el documento HTML.

# Meta

Proporciona metadatos sobre el documento HTML, como el conjunto de caracteres, la descripción, palabras clave, etc.


# Table

## table

Crea una tabla.

## tr

Define una fila en una tabla.

## td

Define una celda de datos en una tabla.

## th

Define una celda de encabezado en una tabla.


# Container

## div

Es un contenedor genérico utilizado para agrupar elementos.

# Line   

## br

Crea un salto de línea en el texto.

## hr

Crea una línea horizontal para separar contenidos.
